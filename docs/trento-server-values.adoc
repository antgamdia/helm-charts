= Trento Server values reference

This page documents every configurable value in `charts/trento-server/values.yaml`.

== Dependencies and interactions

* `trento-web.enabled`, `trento-wanda.enabled`, `trento-mcp-server.enabled`, `postgresql.enabled`, `prometheus.enabled`, and `rabbitmq.enabled` control whether each subchart is deployed.
* `global.rabbitmq.tls.mtls.enabled=true` switches RabbitMQ and clients to mTLS mode.
* `global.rabbitmq.tls.mtls.certManager.*` values are used only when mTLS is enabled and cert-manager resources are managed by the chart.
* `prometheus.server.auth.type` must be set to `none` or `basic`; `prometheus.server.auth.basic.*` is used only in `basic` mode.
* `prometheus.server.ingress.*` applies only when `prometheus.server.ingress.enabled=true`.
